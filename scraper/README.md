# hongkong_flowers

## 1) Data Collection Stage:

### scrape data from python
```
# download to load folder
$ python ./scrape.py -n 100 -f list.csv -p /mnt/scratch/flowers/images/
# or upload to s3
$ python ./scrape.py -n 100 -f list.csv -s
```
